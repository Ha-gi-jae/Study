배열 메소드

indexOf(item, start) : item의 위치를 리턴, start는 옵션(생략가능), 생략시 start를 지정하지 않으면 0부터 시작한다.
lastIndexOf(item, start) : 뒤에서부터 검색 item의 위치를 리턴, 마찬가지로 생략가능 지정하지않으면 마지막부터.

배열 메소드

push(a, b, c) : 배열의 끝에 a, b, c를 추가
pop() : 배열의 끝(마지막)의 요소를 제거

배열 메소드

slice(start, end) : start~end 사이의 요소를 추출해서 새로운 배열을 만든다.
splice(index, n, a, b, c) : index위치에서 n개를 삭제하고 a, b, c 를 추가한다.

배열 메소드

a.concat(b,c) : a배열과 b배열 c배열을 하나로 합친다.
join("/") : 하나의 배열을 문자열로 변환한다.

배열 메소드

reverse() : 배열의 순서 뒤집기
sort() : 사전순으로 나열
