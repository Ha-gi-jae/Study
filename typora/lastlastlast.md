로그인

1. 아이디 존재 여부  확인 
2. 아이디의 비밀번호 일치하는지 확인
3. 아이디의 기관코드 가져오기
4. 로그인정보(*기관코드) 세션에 담아주기. 세션시간 설정.
5. 기관코드(I, P, H, L)에 따라 해당 기관의 메인페이지로.

로그아웃

1. 세션 만료
2. 로그인 페이지로 이동.

아이디 찾기

1. 이름, 생년월일 일치 확인
2. 일치하면 아이디 출력 ( 끝에 2자리는 **로 표시.)

비밀번호 찾기

1. 아이디, 이름, 생년월일, 연락처 일치 확인
2. 일치하면 해당 이메일 또는 연락처에 인증번호 전송
3. 인증번호 확인
4. 비밀번호 변경 페이지로 이동(아이디 가져가야함)

비밀번호 변경페이지

1. 새로운비밀번호 입력(정규식)
2. 새로운 비밀번호 확인(일치여부)

마이페이지

1. 정보 가져오기
2. 출력하기(수정/비밀번호변경)

* 수정버튼 클릭시 수정폼으로 날려줌.
* 비밀번호 변경시 비밀번호 변경 페이지 출력

마이페이지(정보수정폼)

1. 기존 정보 가져오기

2. 등록 시, 입력받은 값 DB에 저장
3. 마이페이지로 이동

세션연장

1. 현재 세션을 30 or 60 분으로 다시 설정
2. JSP 화면에 세션 출력하는곳에 넘겨줘야함.

진단키트

1. 현재 진단키트 수 + 추가등록 진단키트수 DB에 저장
2. 리스트에 반영(추가등록) + reload

피검자 문진표 작성목록

1. CHCK_YN(검사여부) = N  만 조회

피검자 문진표 상세조회

1. PSTI_INNB(피검자고유번호) + INSTT_INNB(기관코드) = 문진표 내용 뽑아오기.

피검자 문진표 등록

1. 검사여부 Y + 기관 고유번호

시료 검사요청

1. 체크박스 선택된거 가져오기.
2. 검사여부 Y DB로 

시료요청 리스트 출력  체크박스 등록버튼
